## 使用方法

#### 下载npm包

```
npm install --save bee-grid
```


#### 样式引入

- 可以使用link引入build目录下bee-grid.css
```
<link rel="stylesheet" href="./node_modules/build/bee-grid.css">
```
- 可以在js中import样式
```js
import "./node_modules/src/Grid.scss"
//或是
import "./node_modules/build/bee-grid.css"
```
